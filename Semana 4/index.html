Aprendiendo CSS3 y Bootstrap 5

Temas de la Materia

HTML5

HTML5 es el lenguaje de marcado para estructurar páginas web. Permite crear contenido semántico y moderno.

HTML5 incluye nuevos elementos como header, footer, article y section.

CSS3

CSS3 nos permite diseñar páginas web con estilos modernos, animaciones y diseños responsivos.

Con CSS3 podemos usar flexbox, grid, transiciones y media queries.

Bootstrap

Bootstrap es un framework que facilita crear sitios web responsivos con componentes predefinidos.

Bootstrap incluye un sistema de grid, componentes como navbar y formularios.
